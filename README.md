# ROOM_MONITOR

## 概要

室内環境をモニタリングするためのモノレポです。

## システム構成

### ■ sensing

室内に設置したセンサーのデータを１分毎に取得し、データベースに保存するシステムです。

| カテゴリー    | 使用技術                         |
| ------------- | -------------------------------- |
| ホスティング  | Cloudflare Workers               |
| データベース  | Cloudflare D1                    |
| 定期実行      | Cloudflare Workers Cron Triggers |
| センサー      | Nature Remo                      |
| ------------- | -------------------------------- |
